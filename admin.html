Flujo del Cliente - De Reserva a Resena

Sigue este flujo con cada cliente para maximizar satisfaccion, resenas y recompra.

1. Reserva Recibida

Enviar mensaje de bienvenida + confirmacion de datos inmediatamente.

2. Cobro / Confirmacion de Pago

Si no ha pagado, enviar link de pago Bold o datos de Nequi/Daviplata.

3. Confirmacion + Punto de Encuentro

Una vez pagado, confirmar cupo y enviar punto de encuentro con hora exacta.

4. Recordatorio (1 dia antes)

Enviar recordatorio con hora, punto de encuentro y recomendaciones.

5. Post-Tour: Agradecimiento (mismo dia)

Agradecer por elegirte, preguntar como la paso.

6. Pedir Resena (1-2 dias despues)

Enviar link directo de Google para que deje 5 estrellas. CLAVE para posicionamiento.

7. Recompra (1 semana despues)

Si sigue en la isla, ofrecerle otro tour con descuento especial de recompra.

Automatizar DMs de Instagram (ManyChat)

Para respuestas automaticas en Instagram DMs, te recomendamos usar ManyChat (plan gratis hasta 1,000 contactos).

Paso 1: Crea cuenta gratis en manychat.com y conecta tu Instagram Business.
Paso 2: Configura palabras clave: cuando alguien escriba "TOURS", "PRECIOS", "INFO" → respuesta automatica con catalogo.
Paso 3: Configura respuesta a comentarios: cuando comenten en un post → DM automatico con link de reserva.
Paso 4: Agrega boton "Enviar mensaje" en tu bio de Instagram para captar leads automaticamente.

ManyChat gratis te permite hasta 1,000 contactos. Perfecto para empezar.
